== Description ==
BUDA completely blocks users who are not administrators from accessing the dashboard, as well as removing the toolbar for non-administrators. Works out of the box, no setup or coding knowledge required.<br>

*Note* - To login to your dashboard, you'll need to login via yourdomain.com/wp-login.php

== Installation ==
1. Upload the plugin's folder to the /wp-content/plugins/ directory, alternatively, just install this plugin from the WordPress plugin's directory
2. Activate the plugin through the plugin's menu in your WordPress dashboard
3. Use yourdomain.com/wp-login.php to login, as /wp-admin will be blocked if you are not logged in
